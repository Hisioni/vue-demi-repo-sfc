# VUE-DEMI-REPO-SFC

> vue-demi + pnpm monorepo + vue sfc

# Environment

> node >= 16 && pnpm >= 7

# Install

```shell
> cd vue-demi-repo-sfc/
> pnpm i
```

# Build

```shell
> pnpm build
```

also you can build along with target version

```shell
> pnpm build:2     -- vue 2.6.x
> pnpm build:2.7   -- vue 2.7.x
> pnpm build:3     -- vue 3.2.x
```
